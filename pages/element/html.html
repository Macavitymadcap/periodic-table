<html>

Description

The <html> tag is one of the most fundamental components of web development, serving as the root element of any HTML document. Its primary purpose is to enclose all other HTML elements, including head and body sections, marking the beginning and end of an HTML document.

The opening <html> tag signifies the start of an HTML document, and the closing </html> tag marks the end. Everything between these two tags is considered part of the HTML document. Before the <html> tag, the <!DOCTYPE html>declaration is included to define the document type and inform the browser that the document is written in HTML5.

Attributes

Includes Global Attributes, but rarely used on the <html> element.

  • lang: The primary language of the document, e.g. en, fr etc. Useful for accessibility and search engine optimization (SEO) purposes. With a languuage attribute set, screen readers and search engines can use this information to provide better content delivery and indexing. A valid lang delaration also ensures the contents of the document's <head> are announced properly.
  • xmlns: The XML Namespace of the document, default http://www.w3.org/1999/xhtml. Required for documents parsed with XML parsers, but optional in HTML5.
