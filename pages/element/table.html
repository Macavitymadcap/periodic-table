<table>

Description

The <table> tag defines an HTML table. An HTML table consists of one <table> element and one or more <tr>, <th>, and <td> elements. The <tr> element defines a table row, the <th> element defines a table header, and the <td> element defines a table cell.

The <table> box establishes a table formatting context. Elements inside the <table> generate rectangular boxes. Each box occupies a number of table cells according to the following rules:

  1. The row boxes fill the table in the source code order from top to bottom. Each row box occupies one row of cells.
  2. A row group box occupies one or more row boxes.
  3. Column boxes are placed next to each other in source code order. A column box occupies one or more columns of table cells.
  4. A column group box occupies one or more column boxes.
  5. A cell box may span over multiple rows and columns. User agents trim cells to fit in the available number of rows and columns.
Table cells do have padding. Boxes that make up a table do not have margins.

Attributes

The <table> element includes Global Attributes.

  • border: Specifies whether the table should have borders.
  • height: Some browsers support a non-standard interpretation of height. The unitless value sets a minimum absolute height in pixels. If set as a percent value, the minimum table height will be relative to the parent container's height. Use the min-height CSS property instead, as this attribute is deprecated.
  • dir: Alters the layout of columns; ltr for left-to-right or rtl for right-to-left.
Summary

Content categories Flow content.
Permitted content In this order:
  1. an optional <caption> element,
  2. zero or more <colgroup> elements,
  3. an optional <thead> element,
  4. either one or more of the following:
    • zero or more <tbody> elements
    • one or more <tr> elements
  5. an optional <tfoot> element
Tag omission None; both start and end tags are mandatory.
Permitted parents Any element that can contain flow content.
Implicit ARIA role table
Permitted ARIA roles Any
DOM interface HTMLTableElement
