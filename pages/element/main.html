<main>

Description

The <main> tag is used to denote the main content of a document. It is intended to contain content that is directly related to the central topic of the document or the central functionality of an application. The content inside the <main> element should be unique to the document and not repeated across other pages.

The tag is strictly informative and does not contribute the document's outline. It has no bearing on on the DOM's concept of the structure of the page. It behaves like a main landmark role, which can be used by assistive technology to quickly identify and navigate to large sections of the document.

The <main> element must not be a descendant of an <article>, <aside>, <footer>, <header>, or <nav> element.

Attributes

The <main> element does not have any unique attributes, but it can include Global Attributes.

  • id Lets the lement be a target of a skip navigation link. This techinque, "skipnav", is a technique that allows an assistive technology user to quickly bypass large sections of repeated content.
