Metadata

Description

Metadata elements provide information about the HTML document itself. They are typically placed within the <head> section and include details such as the document's title, character encoding, linked resources, and scripts.

Elements:
  • <head>
  • <title>
  • <meta>
  • <base>
  • <link>
  • <style>
  • <script>
  • <script>
  • <noscript>
  • <template>
