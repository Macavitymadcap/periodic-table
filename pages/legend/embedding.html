Embedding

Description

Embedding elements are used to incorporate external resources into an HTML document. This includes images, multimedia content, interactive applications, and even other web pages, allowing for rich and diverse content presentation.

Elements:
  • <img>
  • <area>
  • <map>
  • <embed>
  • <object>
  • <param>
  • <source>
  • <iframe>
  • <canvas>
  • <track>
  • <audio>
  • <video>
  • <picture>
  • <portal>
